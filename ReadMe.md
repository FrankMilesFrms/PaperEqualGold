**说明**

本项目基于Android平台开发的软件，最后一次更新代码在2023年。 项目以再生资源回收为开发方向，目的是宣传、提供废纸回收系统的程序。 本项目以“买卖捐赠”四个为主要功能，并由此开发了本项目。
